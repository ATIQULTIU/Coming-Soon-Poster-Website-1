# 🎬 Coming Soon Poster Website

A stylish and responsive **Coming Soon Landing Page** built using pure HTML & CSS.  
Designed with a vintage paper poster aesthetic, masking tape effects, and a clean email signup section.

---

## ✨ Features

- 🎨 Vintage poster style UI
- 📱 Fully responsive design
- 🧷 Realistic tape effects
- 📩 Email signup form
- 🌄 Fullscreen background image
- 🖋️ Custom typography using Google Fonts
- ⚡ Lightweight and fast

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
